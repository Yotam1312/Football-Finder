## Key Decisions

| Decision | Phase | Outcome |
|----------|-------|---------|
| `x-admin-api-key` header (not Authorization Bearer) | 14-01 | Cleaner for cron/script callers |
| Middleware returns 401 when ADMIN_API_KEY env var is unset | 14-01 | Prevents unprotected access in misconfigured envs |
| Hero pill shows time large + date small (replaces VS badge) | 14-02 | Answers who+when at a glance |
| CTA card uses `(ticketUrl \|\| mapsUrl)` single guard | 14-02 | Never renders empty card |
| `transportType` stored as `String?` not enum | 15-01 | Phase 18 enforces at application layer |
| Batched TRANS-02 + TRANS-05 into single migration | 15-01 | One migration file `20260324161018_phase15_transport_schema` |
| Lyft replaced with Bolt | 16-01 | More relevant for European football audience |
| FAQ accordion: one item open at a time | 16-01 | Cleaner mobile UX |
| IIFE pattern for inline JSX destructuring | 17-01 | No hoisted pre-return variable |
| `CircleParking` not `ParkingCircle` | 17-01 | Correct lucide-react@0.577.0 name |
| `hasTransportLines` gated on line arrays only | 17-01 | walkingTime/parkingInfo alone doesn't count as filled |
| `TAB_TO_POST_TYPE` drives both API filter and modal pre-selection | 18-02 | Single source of truth |
| `preSelectedType` not passed when `editPost` set | 18-02 | Edit mode carries its own type, avoids conflict |
